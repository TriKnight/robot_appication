The robot moving with 4 coordinates: A, B, C, D with position below:

| State       | Position[X(m), Y(m)m , orientation(rad)] |
|-------------|------------------------------------------|
| Move A      | (1.3, 1.8, 1.5707)                       |
| Waiting 10s |  10 seconds                              |
| Move B      | (1.3, -1.8, 1.57)                        |
| Waiting 5 s | 5 seconds                                |
| Move C      | (-1.3, -1.8, 1.57)                       |
| Move D      | (1.3, -1.8, 1.57)                        |
